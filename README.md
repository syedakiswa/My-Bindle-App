# MyBindle – Social Platform

A modern social platform where friendships grow and communities connect. Built with Next.js, Tailwind CSS, and Framer Motion for animations.

---

## Features

- Responsive layout for mobile, tablet, and desktop
- Dark & Light mode support
- Smooth animations with Framer Motion
- SEO-friendly with meta tags and semantic HTML
- Accessible (ALT texts, ARIA labels, WCAG-compliant color contrast)
- Optimized images (WebP format)
- Loader with animation
- Carousel for mobile feature cards

---

## Tech Stack

- **Framework:** Next.js (latest)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide Icons
- **Images:** WebP optimized

---


## Hooks Used
- `useState`: Track carousel index & loader visibility
- `useEffect`: Manage loader timing and route changes
- `useRef`: Track first page load to control loader behavior



## Not Implemented / Optional Features

- Dark/Light mode toggle – currently only **light mode** is applied
- Schema markup for advanced SEO (bonus)
- Unit tests for components (Jest / React Testing Library)
